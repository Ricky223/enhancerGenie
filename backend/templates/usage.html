We support bed and bed.gz format. There must be atleast first 3 columns of the bed files without headers, but we recommend 4 columns in sorted order. You can learn more about the bed files here. Here are some test files to get started.

Species Assembly

We only support human species(Human: GRCh38) currently. All our processing are based on GRCh38, using any other species might be processed,but results won't be accurate. You can find more information about the assembly here.
